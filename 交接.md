# 党员思想汇报生成器 - 项目交接文档

## 项目概述

这是一个**党员积极分子思想汇报生成工具**，面向中国投资促进会等机构的入党积极分子使用。

**在线访问**：https://junzhejin443-pixel.github.io/party-report-generator/

## 核心功能

### 1. 思想汇报生成
- 根据用户选择的年月，自动匹配当月时政素材
- 支持选择3种风格：理论学习型、实践结合型、反思成长型
- 生成2-3篇不同版本供选择

### 2. 素材库
- **时事新闻**：9条国内/国际新闻
- **领导讲话**：9条习近平重要讲话
- **党的精神**：9条（伟大建党精神、两个确立、四个自信等）

### 3. 时间递进机制
- 配置首次汇报年月和提交频率（月度/季度/半年）
- 自动计算第N次汇报，阶段递进：
  - 第1-2次：初期（学习体会为主）
  - 第3-4次：中期（融入工作实践）
  - 第5次+：后期（独立思考）

### 4. 去AI化
- 词汇随机替换库（学习→拜读/研读/深入领会）
- 句式变体库（多种开头方式）
- 季节特征词库（新年伊始、阳春三月等）

### 5. 工作内容智能扩展
根据用户输入的工作内容关键词，自动扩展描述：
- 会展/展览 → 策划组织国际投资促进论坛等
- 培训 → 组织专业培训课程等
- 对外联络 → 与各国投资促进机构建立联系
- 国际/外事 → 与外国驻华使领馆联络

## 技术架构

### 单文件HTML应用
- 无后端，纯前端
- 所有JS/CSS/HTML在`index.html`中
- 数据存储在浏览器localStorage

### 文件结构
```
party_report_generator.html  ← GitHub Pages入口（同步自index.html）
index.html              ← 开发主文件（两文件内容相同）
交接.md                  ← 本文档
```

## 官方写作规范

根据[共产党员网范文](https://fuwu.12371.cn/2016/08/19/ARTI1471596869770792.shtml)：

### 格式要求
- 标题：居中写"思想汇报"
- 称谓："敬爱的党组织"
- 结尾："恳请党组织给予批评、帮助"
- 署名："汇报人：XXX" + "XXXX年XX月XX日"

### 内容要求
1. 向党组织如实汇报，敢于亮思想
2. 围绕一两个重点问题谈深谈透，不要罗列多个泛泛而谈
3. 一分为二评价自己，肯定进步，指出不足

## 维护指南

### 添加新的精神提法
在`党的精神提法库`数组中添加对象：
```javascript
{ id: 'spirit10', 名称: '新提法名称', 描述: '具体描述', 推荐月份: [3, 10] }
```

### 添加新的领导讲话
在`领导讲话库`数组中添加：
```javascript
{ id: 'speech10', 来源: '习近平', 讲话: '讲话内容', 推荐月份: [6] }
```

### 修改模板结构
在`生成初期模板_选择模式()`等函数中修改return的模板字符串

### 调整字数设置
修改`根据字数调整内容()`中的扩充素材库

## 部署说明

### GitHub Pages
1. 推送到GitHub仓库
2. Settings → Pages → Source: master branch, / (root)
3. 等待构建后访问 `https://用户名.github.io/仓库名/`

### 注意事项
- `index.html`和`party_report_generator.html`需要保持同步
- 提交前执行`cp index.html party_report_generator.html`

## 问题排查

### 按钮无反应
- 检查浏览器控制台是否有JS错误
- 常见：`Cannot read properties of null (reading 'value')`通常是引用了不存在的DOM元素

### 内容重复
- 检查`随机替换()`函数中的正则匹配
- 连续重复词清洗：`result.replace(/([一-龥])\1{2,}/g, '$1$1')`

### 版本差异不明显
- 增强`版本策略微调`函数中的替换逻辑
